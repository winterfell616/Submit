1. "WAR" folder contains the files to run the task. Copy the "messagescheduler" folder to "webapps" directory of a server that can serve up static web pages. The author was using TC Server v3.0 which is powered by tomcat-8.0.15.
   Type http://localhost:8080/messagescheduler/ (adjust the port according to your configuration) into your browser to run the application.

2. "CODE" folder contains all the source code and testb cases written to support the work.
